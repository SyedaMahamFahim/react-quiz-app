# Tech Stack
- React 
- Axios
- 3rd Party Api Integration
## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/react-quiz-app
```

Go to the project directory

```bash
  cd react-quiz-app

```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
